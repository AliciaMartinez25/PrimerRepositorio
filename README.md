## Explicación de la tarea

Este proyecto tiene como objetivo familiarizarse con GitHub y con la comunidad tecnológica en redes sociales.

### Apartado 1: GitHub y "Hola Mundo"
- Crear una cuenta en GitHub.
- Generar un primer repositorio que incluya:
  - Un archivo **README** con una breve explicación de la tarea escrita en Markdown.
  - Un archivo sencillo **"Hola Mundo"** programado en Python.

### Apartado 2: Conexión con la comunidad
Según la red social utilizada:
- **Twitter/X**:
  - Crear una lista relacionada con la inteligencia artificial.
  - Añadir perfiles que compartan contenido sobre IA.
  - Seguir listas sobre lenguajes de programación como Python, R o C++.
- **LinkedIn**:
  - Seguir a *Comunidad Spain AI*.
  - Buscar y seguir a profesionales del ámbito de la inteligencia artificial.
